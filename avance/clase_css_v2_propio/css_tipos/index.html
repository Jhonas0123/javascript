primera forma de usar css

dentro de la misma etiqueta que queremos cambiar ejmplo dentro de las mismas etiquestas de h1, p, section, button y otros.

segunda forma de usar css

con la etiqueta "style", dentro del head usamos esta etiqueta y ahi hacemos los llamados a las etiquetas de proyecto, este si es mucho codigo nos ocacionara problemas a la hora la legibilidad del codigo.

tercera forma de usar css

en este caso creamos un archivo que este sea style.css y dentro de este hacemos todas las configuracion de css, sin olvidar que desde el index.html tenemos que hacer llamado a styles.css con la etiqueta link y este debe ir dedntro del head
