existe la posibilidad de dar estilo de manera generica a las etiquetas pero no es buena practica y para esto existe 2 tipos: por (clases) y por (ID)

para (CLASE), nota para hacer llamado es con punto (.) y luego el nombre de la clase

para (ID), nota para hacer llamado es con numeral (#) y el nombre del ID

La diferencia entre estos dos es que la (clase) se puede replicar en cualquier etiqueta pero el (id) es unico. Por que ocurre esto? esto pasa por que a la hora de trabajar con javascript esto nos provocara problemas.
